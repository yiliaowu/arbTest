# 🚀 LOF基金套利监控系统 - 新手入门与运行指南

欢迎来到 LOF 基金套利系统！对于刚接手这个项目的新手，你可能会对眼前的一堆 Python 脚本和黑色的终端窗口感到一丝困惑。别担心，这篇文档将带你从零开始，了解这套系统是如何像施了魔法一样自动运转的。

## 🌟 1. 系统核心理念：分离代码与数据

在传统的项目中，分享代码往往连带着庞大且私密的数据库文件一起分享。但在这个项目中，我们采用了**“现代化开源项目”**的标准结构：
- **只分享代码，不分享数据**。
- 无论谁在自己的电脑上下载了这份代码，只要双击运行，系统就会**自动在本地从零开始搭建数据库**，并自动去互联网上把最新的数据爬取下来填充进去。

这不仅保护了数据隐私，也让你拥有了一个干干净净的、完全属于自己的本地数据库。

## 🛠️ 2. 首次启动：见证“魔法”建库

当你通过 Git 克隆（Clone）或者下载了这份代码后，你的电脑里**没有**任何数据库文件（没有 `.db` 结尾的文件）。

### 操作步骤：
1. 确保你的电脑上已经安装了 **Python 3**，并且配置好了环境变量（通常安装时勾选 `Add Python to PATH` 即可）。
2. 进入 `LOFarb` 文件夹。
3. 双击运行 **`LOF_start_lof_system.bat`**（一键启动脚本）。

### 后台发生了什么？
当你双击运行后，不要关闭弹出的黑色窗口，观察里面的滚动文字，系统正在按顺序执行以下操作：
1. **清理环境**：杀掉可能卡死的旧 Python 进程，释放 5000 端口。
2. **`[0/6] 运行 011`**：这是**最神奇**的一步。底层程序发现你的电脑里没有数据库，会瞬间在 `database/` 目录下**自动创建一个 `arb_master.db` 数据库**，并建好几十张空表。接着，它化身爬虫，去新浪、东财等网站把今天的最新行情、汇率全部抓取下来，塞进这个刚建好的空库里。
3. **`[1/6] 运行 012`**：计算大脑启动。它读取刚才 `011` 抓下来的原始数据，经过复杂的数学推演，计算出每个 LOF 基金的“静态官方估值”。
4. **启动服务 (`02` 和 `03`)**：启动网页服务器（端口 5000 和 5002），生成今天最新的 HTML 监控看板，并**自动在你的浏览器中打开**！

只要黑窗口里没有致命报错，你的系统就成功满血复活了！

## 🔍 3. 常见问题：为什么提示“缺失数据”？

因为你的数据库是“全新出厂”的，它虽然抓取了最新的数据，但**缺少历史数据**。

在运行 `012` 计算脚本时，你可能会在黑色控制台里看到类似这样的黄色警告（探针报警）：
```text
  🕵️ [数据探针] 易方达纳100(161130) 最近3个净值日的数据对齐情况:
    ⚠️ 2024-04-30 (NAV=4.188): 缺失 -> 人民币中间价
```

### 为什么会报警？
系统的估值计算非常严谨。比如计算纳指 LOF (161130) 的估值时，不仅需要美股的指数，**强烈依赖当天的“人民币中间价”**。由于新建的数据库没有前几天（比如 4月30日）的历史汇率，程序为了防止算出错误的利润，宁可报警并拒绝计算当天的历史估值。

### 怎么解决？（使用历史数据导入脚本）
已经为你准备好了一份历史汇率的 Excel 文件。我们只需要运行一个辅助脚本把它“喂”给数据库即可。

**操作步骤：**
1. 确保你已经安装了处理 Excel 的库（如果没装，在终端运行 `pip install pandas xlrd lxml html5lib`）。
2. 找到群里提供的汇率 Excel 文件（例如 `人民币汇率中间价 0506.xls`）， 最佳的办法自己去https://www.safe.gov.cn/safe/rmbhlzjj/index.html 程序下载
3. 运行项目提供的专门导入脚本 `import_fx.py`（该脚本会自动读取 Excel 并写入数据库）。
4. 看到 `🎉 导入完成` 后，在你的浏览器维护面板（端口 5002）里点击 **[重新计算估值]**（或者重新运行一次 `LOF_start_lof_system.bat`）。
5. 此时 012 计算大脑会发现历史汇率齐备了，完美算出历史估值！

## 🖥️ 4. 认识我们的“三驾马车” (核心模块)

在项目源码中，有几个频繁出现的文件名前缀，了解它们的作用，你就掌握了系统命脉：

*   **`LOF011_daily_updater.py` (数据搬运工)**：纯爬虫。负责把互联网上的杂乱数据（汇率、美股、A股、Woody因子等）搬回家，存入 SQLite 数据库。每天运行一次即可。
*   **`LOF012_calculate_static_valuation.py` (精算大脑)**：数学家。不联网，只对着本地数据库里的数据发呆，算出 T-1 的静态估值、对冲份额、历史溢价率，然后把结果写回数据库。
*   **`LOF02_fetch_trade_data.py` (实时心脏)**：发动机。提供 5000 端口的 Web 服务，连接 IB 外盘、QMT/通达信 A 股极速行情，将毫秒级的盘口跳动通过 WebSocket 推送给前端网页。
*   **`LOF03_*.py` 系列 (前端画师)**：负责把枯燥的数字渲染成你看到的那个漂亮、带有红绿闪烁灯的 HTML 看板页面。

## 💡 5. 日常使用小贴士

1.  **我每天需要做什么？**
    每天早上打开电脑，双击运行一次 `LOF_start_lof_system.bat` 就可以了。它会自动帮你完成签到、更新数据、开启面板的所有流程。
2.  **如果白天 A 股价格不跳动了怎么办？**
    看一下看板顶部的 `A股实时行情` 状态，如果是断开，可以点击旁边的 `🔄 尝试重连` 按钮。
3.  **我想修改基金的基础仓位怎么办？**
    访问底部的 `LOF基金配置中心` (默认在 5001 或 5002 端口)，在网页里通过图形化界面修改保存即可，它会自动修改后端的 YAML 文件，无需手写代码！
